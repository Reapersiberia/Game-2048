# Игра 2048

Классическая игра 2048, реализованная на HTML, CSS и JavaScript.

## Описание

2048 - это логическая игра-головоломка, в которой цель состоит в том, чтобы объединять плитки с одинаковыми числами, чтобы получить плитку с числом 2048.

## Как играть

- Используйте стрелки на клавиатуре (↑ ↓ ← →) для перемещения плиток
- Когда две плитки с одинаковым числом соприкасаются, они объединяются в одну
- После каждого хода появляется новая плитка (2 или 4)
- Игра заканчивается, когда сетка заполнена и нет возможных ходов
- Цель - достичь плитки 2048!

## Управление

- **Стрелки клавиатуры**: Перемещение плиток
- **Сенсорный экран**: Свайпы в нужном направлении
- **Кнопка "Новая игра"**: Начать новую игру

## Технологии

- HTML5
- CSS3
- Vanilla JavaScript

## Запуск

Просто откройте файл `index.html` в вашем браузере.
